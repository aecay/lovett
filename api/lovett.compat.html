lovett.compat module¶

Functions to interoperate with other parsed corpus software.

strip_corpussearch_comments(s)[source]¶

Remove CorpusSearch style comments from a string.

This function removes two types of comments from a string. The first is generated by CorpusSearch, and takes the form of blocks delimited by /* ... */ and /~* ... *~/. The second are single line comments of the form <+ ... +> generated by emacs parser-mode (or some other awful relic).

Parameters:s (str) – the string to operate on.
Returns:s with comments stripped.
Return type:str
